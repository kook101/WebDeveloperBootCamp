Boolean Logic

Everything starts with the idea that a statement is either True or False.
Then we can combine those initial statements to create more complex statesments that also evaluate to True or False Comparison Operators.

Assuming x=5

Operator Name Example Result
> Greater than x > 10 false
>= Greater than or equal to x >= 5 true
< Less than x < -50 false
<= Less than or equal to x <= 100 true
== Equal to x == 5 true
!= Not equal to x != b true
=== Equal value and type x === 5 false
!== Now equal value or equal type x !== 5 true
